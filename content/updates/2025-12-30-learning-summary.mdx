最近学习了更多关于 Next.js 与 Vue 3 的知识：
- OptionS 和 Components 的区别
- 动态路由和静态生成
- 中间件的使用
- 引用外部CDN的壁纸资源，如[SOMEACG](https://someacg.vercel.app/) （需要魔法访问）

以及补足我最短板的内容：
- Vue 3 的 Composition API 写法
- Options API 和 Composition API (类似Hooks) 的区别
- 响应变：defineProperty → Proxy 
- 性能变：虚拟DOM优化 + Tree-shaking

比如 这里是一个 获取用户数据并展示 的例子：
```vue
<template>
  <div>
    <h1>用户列表</h1>
    <ul>
      <li v-for="user in users" :key="user.id">
        {{ user.name }} - {{ user.email }}
      </li>
    </ul>
  </div>
</template>
```


这些知识对构建高性能的 Web 应用非常有帮助！
